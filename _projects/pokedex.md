A semantic web-based approach to implement a knowledge representation system related to "Pokemon". We linked different sources including several external datasets, designed the ontology, and constructed the knowledge graph using knowledge representation tools for the web.

**Key Features:**
- Integrated multiple external datasets
- Designed comprehensive ontology
- Built a knowledge graph for Pokemon information
- Implemented advanced querying capabilities

The system has a clear structure that allows users to easily search for Pokemon information and perform complex queries across different data sources.
